# Пицца Тагил — сайт (React + TS + Vite + Tailwind)

## Запуск
```bash
npm install
npm run dev
```

## Сборка
```bash
npm run build
npm run preview
```

## Cloudflare Pages (SPA)
- Build command: `npm run build`
- Output directory: `dist`
- Для SPA-роутинга добавь файл `public/_redirects` со строкой:
  `/* /index.html 200`

(Файл уже добавлен в этот проект.)
